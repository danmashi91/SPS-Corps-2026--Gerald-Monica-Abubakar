# Local Agentic Loan Triage System

## Overview
This project implements a local, CPU-compatible, agentic LLM-powered decision-support system for structured loan application triage.

The system assists lending operations staff by:
- Evaluating structured applicant data
- Applying deterministic risk scoring
- Interpreting lending policy constraints
- Generating structured triage recommendations
- Logging decision rationale locally

This system does NOT issue final loan approvals.

## Project Goals
- Build a fully local LLM-powered agent
- Maintain strict offline capability
- Provide structured explainable recommendations
- Evaluate performance using measurable metrics
